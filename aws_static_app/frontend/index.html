Serverless Architecture on AWS

Architecture du déploiement

Utilisateur → CloudFront (CDN) → S3 (static files)
API Gateway → Lambda → IAM Roles

S3 Static Hosting

Bucket configuré en hébergement de site statique public, servant les fichiers HTML/CSS/JS.

CloudFront CDN

Distribution du contenu via les edge locations AWS pour réduire la latence globale.

EC2 (Ubuntu)

Instance configurée avec accès SSH sécurisé et rôle IAM dédié pour les opérations backend.

Lambda + API Gateway

Architecture RESTful serverless — exécution à la demande, sans serveur à maintenir.

Architecture documentée — Free Tier cost-optimized
